# ✦ xPsMenu ✦
*Developed by xPlugins*

## ☠ Overview
A fully translated, deeply aesthetic, and highly customizable protection menu system. Redesigned with a clean, dark, and aggressive aesthetic to fit modern server standards.

## ✦ Features
- **Full English Translation:** Every menu, flag, and message is localized.
- **Dark Aesthetic:** Custom hex color implementation (`#FF0000`, `&f`, `&8`) combined with black stained glass UI.
- **Live Reload:** Make changes on the fly without restarting your server.
- **Custom Particles:** Uses `ASH` and `DRAGON_BREATH` to fit the dark theme.

## ⋆ Installation
1. Drop the configuration files into your `plugins/xPsMenu` folder.
2. Run `/psmenu reload` in-game or from the console.
3. Enjoy the new interface!
